# Cafe-Wifi

A Flask web application for managing and displaying a directory of cafes with information about their WiFi, amenities, and more.

## Features
- Add new cafes
- Edit existing cafe details
- Delete cafes
- View a list of all cafes with detailed information

## Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS (Bootstrap)
- WTForms

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Cafe-Wifi.git
   
2.   Navigate to the project directory:
cd Cafe-Wifi

3. install dependencies:

pip install -r requirements.txt

4. Create a .env file with the following:
   
SECRET_KEY=your-secret-key
DATABASE_URL=sqlite:///cafes.db


5. Run the app:
   flask run
